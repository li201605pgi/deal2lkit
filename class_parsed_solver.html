Detailed Description

template<typename VECTOR>
class ParsedSolver< VECTOR >

A solver selector which uses parameter files to choose between different options. This object is a LinearOperator which can be called in place of the inverse of another LinearOperator.

Example usage is the following:

ParsedSolver<VEC> Ainv;
ParameterAcceptor::initialize(...);
Ainv.op = linear_operator<VEC>(A);
Ainv.prec = linear_operator<VEC>(sompreconditioners);
x = Ainv*b;

Constructor. Build the inverse of an Operator using a parameter file. A section name can be specified, the solver type, the maximum number of iterations, and the reduction to reach convergence. If you know in advance the operators this object will need, you can also supply them here. They default to the identity, and you can assign them later by setting op and prec.
